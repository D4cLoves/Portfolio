# Portfolio — Vladislav Trubnikov

Лендинг-портфолио разработчика.

**Демо:** https://portfolio-puce-one-38.vercel.app

## Стек

- HTML5, SCSS (BEM), TypeScript
- Vite (сборка)
- Vercel Serverless Functions (API)
- Nodemailer (отправка писем)
- Groq API + Llama 4 (AI-чат)

## Запуск

```bash
npm install
npm run dev
```

## Форма обратной связи

Поля: имя, телефон, email, комментарий.

Клиент:
- Валидация всех полей перед отправкой
- Inline-ошибки под каждым полем
- Состояния: loading → success / error
- Блокировка кнопки при отправке

Сервер (`api/contact.ts`):
- Повторная валидация на бэкенде
- Отправка письма владельцу сайта
- Отправка копии пользователю
- JSON-ответ с результатом

## AI-интеграция

Чат-виджет в правом нижнем углу — можно задать вопрос обо мне, AI ответит на основе моих данных. Работает через Groq API.
